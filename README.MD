# AutoShop Backend

This is the backend API for the **AutoShop** project. It provides endpoints to manage categories, spare parts, and sales for an auto parts store. Built with **FastAPI** and **SQLAlchemy**, using **SQLite** as the database.

## Features

- Add, view, and manage **categories**
- Add, view, and manage **spare parts**
- Record and view **sales**
- RESTful API endpoints
- SQLite database integration

---

## Tech Stack

- **FastAPI** – Backend framework
- **SQLAlchemy** – ORM for database interactions
- **SQLite** – Database
- **Pydantic** – Data validation
- **Uvicorn** – ASGI server
